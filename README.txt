Rakenduse nimi:
CozyPlayer

Eesmärk:
Luua multimeediarakendus .NET MAUI-s, mis näitab minu teadmisi mobiilirakenduste arendamisest.

Funktsioonid:
- Helifailide esitamine  
- Lugude salvestamine SQLite andmebaasi  
- Drag & Drop funktsioon lugude järjekorra muutmiseks  
- Mitmekeelne tugi (nt eesti ja inglise keel)  
- Teema vahetamine (hele/tume režiim)  
- Kasutaja eelistuste salvestamine  

Tehnoloogiad:
- .NET MAUI  
- MVVM arhitektuur  
- SQLite-net-pcl  
- Plugin.Maui.Audio  
- Preferences API  
- .resx lokaliseerimine

Mis töötab, mis jäi pooleli
Muusika mängib, aga katkestusega. Proovisin parandada seda, aga praegu tulemused ei ole.
Töötab teemade ja keelede valik, failide lisamine, drag n drop, dünaamilised sätted.
